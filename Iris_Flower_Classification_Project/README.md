
# 🌸 Iris Flower Classification Project

## 📌 About the Project
This is a simple Machine Learning project that predicts the type of Iris flower based on input features like sepal length, sepal width, petal length, and petal width.

The model learns patterns from the dataset and then predicts the correct flower species.

---

## 📂 Dataset Information
We are using the **iris.csv** dataset which contains:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Target column)

---

## ⚙️ Workflow of Project

1. Load dataset using pandas
2. Check and understand the data
3. Separate input and output:
   - X = features (input)
   - y = species (output)
4. Convert species names into numbers (encoding)
5. Split data into training and testing sets (80% train, 20% test)
6. Create Logistic Regression model
7. Train the model
8. Test the model
9. Calculate accuracy

---

## 🧠 Machine Learning Model Used
- Algorithm: Logistic Regression
- Library: Scikit-learn
